# Informe LaTeX

Informe escrito para explicar la limpieza, las variables, los gráficos, las correlaciones y las decisiones tomadas.

## Archivos

- `main.tex`: código fuente del informe.
- `informe_limpieza_datos_mineria_chilena.pdf`: PDF compilado.
- `tablas/`: tablas generadas desde los CSV limpios.
- `figuras/`: gráficos generados desde el notebook.

## Compilar

Desde esta carpeta:

```bash
tectonic main.tex
```
