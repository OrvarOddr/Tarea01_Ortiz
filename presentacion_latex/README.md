# Presentación LaTeX

Presentación Beamer usada para exponer la Tarea 1.

## Archivos principales

- `main.tex`: código fuente de la presentación.
- `main.pdf`: presentación compilada.
- `imagenes/`: imágenes externas descargadas para la temática minera.
- `graficos/`: gráficos generados desde el notebook.
- `fuentes_imagenes.md`: enlaces y créditos de las imágenes externas.

## Compilar

Desde esta carpeta:

```bash
tectonic main.tex
```

Si se usa una distribución LaTeX tradicional:

```bash
pdflatex main.tex
pdflatex main.tex
```
